bashblog ของกระผม

Date: 2025-11-30 - นายเพนกวิน

ก่อนหน้าไม่นานี้ผมได้ลองใช้ bashblog ของ cfenollosa อย่างจริงจังในการเขียนบล็อก แต่ปรากฏว่าเกิดปัญหาในการสร้างแท็ก ซึ่งผมเองพยายามในการแก้ไข แต่ก็ไม่สำเร็จ ดังนั้นกระผมจึงสร้างไฟล์ bash ของผมเองเลย โดยใช้ AI ในการช่วยเขียน แต่ทั้งนี้ทั้งนั้นผมยังชอบ basblog ของ cfenollosa โดยกระผมมีการปรับดังนี้

  1. สร้างไฟล์จาก .md ที่สร้างต่างหาก โดยที่ชื่อไฟล์สามารถตั้งได้เอง และจะอยู่ในโฟลเดอร์ /data ที่ไ่มได้สร้างผ่าน Terminal โดยยึดโครงสร้างเดิม คือ บรรทัดแรกเป็นชื่อเรื่อง ต่อมาจะเป็นเนื้อหา และสุดท้ายจะเป็นแท็ก โดยกระผมใช้การใส่ข้อความลงในเครื่องหมาย [...] สำหรับการสร้างแท็ก
  2. การรันไฟล์ bash ไม่ต้องมีพารามิเตอร์ใด ๆ ทั้งสิ้น โดยเมื่อรันระบบจะลบไฟล์ทั้งหมดใน /html และทำการสร้างใหม่
  3. สามารถสร้างวันที่ย้อนหลังได้โดยการพิมพ์ DATE: ... ตามด้วยปี(4 หลัก) เดือน วัน หรือ/และเวลา (YYYY-MM-DD HH:mm) และเมื่อรันไฟล์ bash ระบบจะทำการจัดเรียงตามวันที่ให้อัตโนมัติ
  4. การพิมพ์เนื้อหาสามารถพิมพ์ได้โดยใช้ markdowm และเราจะใช้ markdown-it ในการแปลงเป็น html

ภาพรวมจะเป็นดังนี้

/data/*.md
   ├─ อ่านชื่อเรื่อง / วันที่ / แท็ก
   ├─ แปลง Markdown → HTML
   └─ เก็บใน POST_LIST + TAG_MAP
       ↓
สร้าง HTML ของแต่ละโพสต์ → html/post.html
สร้าง index.html → เรียงโพสต์ตามวันที่
สร้าง tags.html + tag_xxx.html
สร้าง rss.xml

สิ่งที่จะทำต่อไปคือ การจัดระเบียบให้สวยงาม เพิ่มลิงก์ของแท็กทั้งหมด หน้าอ่านทั้งหมด และการเรียงเป็นเดือนที่ชัดเจนกว่านี้
