การเกิดข้อผิดพลาด ‘bash\r’

Date: 2025-12-14 - นายเพนกวิน

เกิดข้อผิดพลาด /usr/bin/env: ‘bash\r’: No such file or directory /usr/bin/env: use -[v]S to pass options in shebang lines ปัญหานี้เกิดจาก ไฟล์มี \r (CRLF) แบบ Windows

ทำให้ Shebang กลายเป็น #!/usr/bin/env bash\r

Linux จะอ่าน bash\r เป็นชื่อไฟล์ → หาไม่เจอ → error วิธีแก้ใช้คำสั่งนี้:

sed -i 's/\r$//' build.sh
