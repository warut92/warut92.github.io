การติดตั้ง Markdown-it
DATE: 2025-11-30

เนื่องจากผมได้สร้างไฟล์ bash สำหรับการสร้างบล็อกใหม่ และได้ใช้โปรแกรมแปลง markdown เป็น html ใหม่ด้วย จาก Markdown.pl เป็น markdown-it จาก node.js 

มาดูว่าวิธีการติดตั้งเป็นอย่างไร
1. `sudo apt update`
1. `sudo apt install nodejs npm`
1. `npm install markdown-it`

หรือ
1. `sudo apt install python3-markdown-it`

เท่านี้เป็นอันเสร็จสิ้น
