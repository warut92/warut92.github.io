การแก้ปัญหาในข้อผิดพลาด

Date: 2025-12-12 13:48 - นายเพนกวิน

ในการเรียกใช้งานคำสั่งของไฟล์ bash หากเกิดข้อผิดพลาด ดังนี้ /usr/bin/env: ‘bash\r’: No such file or directory /usr/bin/env: use -[v]S to pass options in shebang lines

ให้ใช้คำสั่ง sed -i 's/\r$//' <ชื่อไฟล์>.sh
