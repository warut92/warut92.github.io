การใช้ Git ผ่าน Terminal

1. เมื่อเพิ่มไฟล์ใหม่ ให้ใช้คำสั่ง `git add .`
2. จากนั้นให้ดูสถานะว่าเป็นอย่างไรด้วย `git status`
3. จากนั้นบันทึกข้อความสำหรับการส่งเข้า git ด้วย `git commit -m 'ข้อความ'`
4. บันทึกเข้า git ด้วย `git push` เทอร์มินอลจะถามชื่อผู้ใช้กับรหัสผ่าน ถ้าถูกต้องก็เป็นอันเสร็จ
